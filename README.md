# Differential Graph Analysis for IoT Devices using LEA Encryption Algorithm

## The code designed for this work will be available once the final paper is published

### 📄 Abstract (summary)
Leading cryptanalysis techniques use a Difference Distribution Table (DDT) that provides minimal details for training Machine Learning (ML) models. The code in this repository will present a method to extend the DDT and transform it into a feature-rich dataset with enriched structural details that can be used to train ML models. The ML models are then used to efficiently map and navigate the differential space of LEA.

Information security — Lightweight cryptography Part 2: Block ciphers ISO guidelines are available at [ISO/IEC 29192-2:2019](https://www.iso.org/standard/78477.html)
